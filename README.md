# Emerald Factory — Interactive Site Map

A self-contained, interactive site map of the Emerald factory complex with
per-factory IoT-status floor plans. Click any building on the homepage to open
its floor plan; every machine is colour-coded by IoT status.

## Status legend

| Colour | Meaning |
|--------|---------|
| 🟢 **Green** | IoT already installed |
| 🔴 **Dark red** | Machine ≥ 10 years old (not a candidate) |
| 🟥 **Light red** | IoT can be installed |
| ⬜ **Grey** | Inactive / no current data |

## Notes

- Every page is fully self-contained — floor-plan images are embedded, so the
  files work offline and require no external assets.
- Hosted on GitHub Pages; updates pushed to the `main` branch go live in ~1 minute.
